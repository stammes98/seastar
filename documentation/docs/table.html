Running the Wave Table

This section outlines how to run and troubleshoot the wave table code

Return home

The Wave Table Code

The code is located in the folder "/home/seastar/seastar_ik/wave table" - the program you'll want to run is "python table.py"

To change the speed of the motor, change the second parameter of the function call on line 7, this value should be in RPM. Be careful, it's already multiplied by 10 for the gear box ratio!

If it throws errors

One of the drivers that teknic motors uses loves to delete itself for some reason. Thankfully this is easy to re-install. Navigate the terminal to the folder "/home/seastar/install/teknic/2023_download/ExarKernelDriver" and execute the following commands

  • sudo su
  • ./Install_DRV_SCRIPT.sh
