The long way

Several steps are needed to be executed in a specific order. All terminal windows should be in the main workspace, i.e. /steve_summer_workspace

    Terminal 1
  • source install/setup.bash
  • ros2 run motor_state_manager motor_state_manager
    • Terminal 2
  • source install/setup.bash
  • ros2 run ik_solver ik_solver
    • Terminal 3
  • source install/setup.bash
  • ros2 run ik_tools ik_tools
    • Terminal 4
  • source install/setup.bash
  • ros2 run solar_ephemeris solar_ephemeris
    • Terminal 5
  • source install/setup.bash
  • ros2 run robot_localization ekf_node
    • Terminal 6
  • source install/setup.bash
  • ros2 run bluespace_ai_xsens_mti_driver xsens_mti_node
    • Terminal 7
  • source install/setup.bash
  • ros2 run sunspotter sunspotter
    • Terminal 8
  • source install/setup.bash
  • ros2 run head_chef head_chef
If the robot freezes at this point, go to terminal 6, ctrl-c the program, and restart it.
