Motor State Manager Node

Handles motor connections in pairs of 2. Indexing from zero, each pair of motors will listen to commands and publish its position on numbered topics.

Return home

Motor kinematics restrains are stored in motor_state_manager/include/motorsettings.cfg

Subscribed topics

  • "/motor_commands" - a string formatted as "MOTOR_INDEX V1,V2" which sets velocities of each pair.

Published topics

  • "/motorsN" - N is index of motor pair. Publishes positions of motor pair in stepper space.
