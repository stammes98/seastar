Message Types

  • DesiredPointing
    • float64 azimuth - azimuth of target in degrees
    • float64 altitude - altitude of target in degrees
    • float64 polarization - polarization angle of target in degrees. Motor zero has yet to be defined, will likely need an offset.
  • MotorCommand
    • float64 q1 - position of roll motor in degrees
    • float64 q2 - position of pitch motor in degrees
    • float64 q3 - Position of the stokes motor in degrees
  • SkyEphemeris
    • float64 azimuth - azimuth of target in degrees
    • float64 altitude - altitude of target in degrees

Service Types

  • PointMotors
      Input:
    • DesiredPointing point - target pointing for IK to get to/li>
    • MotorCommand input_state - Current positions of all the motors. Allows for fastest solution to target, or staying at current position if target can't be reached.
      • Output:
    • MotorCommand command - Response from ik_solver, will either be desired pointing, or current position if no solution could be found.
