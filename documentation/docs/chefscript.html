ChefScript

ChefScript is the scripting language the head_chef node reads and converts to commands. Below is documentation of all the commands

Return home

Command list

Command - Long name - Description - Parameters
    • HOAL - HOMEALL
    • Homes the motors against their hardstops
      • Parameters:
    • None.
    • HOON - HOMEONE
    • Homes the motors against their hardstops
      • Parameters:
    • Index of motor to home. 0 = roll, 1 = pitch, 2 = stokes, 3 = plate
    • FISU - FINDSUN
    • Points SeaSTAR towards IMU + IK determined location of the Sun
      • Parameters:
    • None.
    • FSSU - FINDSTAYSUN
    • Points SeaSTAR towards IMU + IK determined location of the Sun, and stay there for a set amount of time
      • Parameters:
    • Time to hold in ms
    • TRSU - TRACKSUN
    • Uses the camera to follow the brightest spot on the sky, and publish the difference between the calculated and actual pointings.
      • Parameters:
    • Time to track in ms
    • CRSU - CIRCLESUN
    • Do a sky scan around the sun
      • Parameters:
    • Size of circle around sun in degrees offset
    • SUAO - SETSUNAZOFF
    • Set an offset to the solar azimuth
      • Parameters:
    • Solar azimuth offset in degrees
    • SUEO - SETSUNALTOFF
    • Set an offset to the solar altitude
      • Parameters:
    • Solar altitude offset in degrees
    • SATG - SETAZTARG
    • Set the secondary target azimuth
      • Parameters:
    • Azimuth of secondary sky target in degrees
    • SETG - SETALTTARG
    • Set the secondary target altitude
      • Parameters:
    • Altitude of secondary sky target in degrees
    • SSTG - SETSTOKETARG
    • Set the secondary target stokes angle
      • Parameters:
    • Stokes angle of secondary sky target in degrees
    • SSSU - SETSUNSTOKE
    • Set the solar target stokes angle
      • Parameters:
    • Stokes angle desired when pointing at / around the Sun in degrees
    • FTRG - FINDTARG
    • Use the IMU / IK to point at the secondary target
      • Parameters:
    • None
    • FSTG - FINDSTAYTARG
    • Use the IMU / IK to point at the secondary target and stay there for a set time
      • Parameters:
    • Time to stay in ms
    • CRTG - CIRCLETARG
    • Do a sky scan around the secondary target
      • Parameters:
    • Radius of sky scan in degrees offset
    • ARCS - ARCSTART
    • Set start of arc for partial scan range
      • Parameters:
    • Position on circle to start partial sky scan at, in degrees
    • ARCE - ARCEND
    • Set end of arc for partial scan range
      • Parameters:
    • Position on circle to end partial sky scan at, in degrees
    • ARSU - ARCSUN
    • Do a scan from the arc start to arc end around the Sun
      • Parameters:
    • Radius of arc
    • ARTG - ARCTARG
    • Do a scan from the arc start to arc end around the secondary target
      • Parameters:
    • Radius of arc
    • GOTO - GOTO
    • Return to a certain line in the code. One indexed
      • Parameters:
    • Line of code to return to
    • NOOP - NO OP
    • Does nothing. Placeholder.
      • Parameters:
    • None

WARNING

DO NOT INCLUDE BLANK LINES! The program thinks an empty line is a command, and struggles with an empty command.
