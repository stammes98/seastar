IK Solver

IK Solver is a ROS service that takes in two custom message types, and responds with one custom message. The custom messages are better documented in SeaSTAR interfaces.

Return home

The service exists on "/ik_solver" topic. Uses modified LM2 method from pykin library to solve IK without positioning error, just directional error. Node contains extra helper script called ik_helper.py, which contains re-written version of pykin functions that ignore position.

Input messages

  • DesiredPointing
    • float64 azimuth - target azimuth for IK to point at in degrees
    • float64 altitude - target altitude for IK to point at in degrees
    • float64 polarization - roll relative to the sky, in degrees, in reference to as-of-yet undetermined vertical angle
  • MotorCommand
    • float64 q1 - current position of the roll motor in degrees
    • float64 q2 - current position of the pitch motor in degrees
    • float64 q3 - current position of the stokes motor in degrees

Response message

  • MotorCommand
    • float64 q1 - target position of the roll motor in degrees
    • float64 q2 - target position of the pitch motor in degrees
    • float64 q3 - target position of the stokes motor in degrees
If the IK solver fails for some reason, it will respond with the position it was given at the start, as to not freak the motors out.
